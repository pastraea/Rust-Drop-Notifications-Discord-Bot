# Support and Maintenance

## Status

This project is actively maintained on a best-effort basis.

## What to Expect

- Bug fixes and dependency updates are prioritized over new features.
- Feature requests are considered based on scope, safety, and maintenance cost.
- No uptime SLA is provided.

## Getting Help

- Open a GitHub issue for bugs or questions.
- Use the issue templates to provide reproducible details.
- Include logs, environment variables used (without secrets), and command outputs.
